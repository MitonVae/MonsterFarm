# ⌨️ 界面与操作

## 整体布局

游戏界面由以下几个区域组成：

```
┌──────────────────────────────────────────────────────┐
│  左侧快捷栏   │           主内容区域              │
│  （固定）     │   （标签页切换：农场/怪兽/探索...）│
│              │                                    │
│              │                                    │
│              │                          [悬浮球]  │
└──────────────────────────────────────────────────────┘
```

---

## 左侧快捷栏

左侧栏提供最常用操作的快捷入口，从上到下依次为：

| 按钮 | 图标 | 功能 |
|------|------|------|
| **手动存档** | 💾 | 立即将当前游戏状态保存到本地 |
| **召回存档** | 📂 | 读取上一次的存档恢复游戏状态 |
| **教程** | 📖 | 打开新手教程面板 |
| **更新日志** | 📋 | 查看版本更新记录（有红点提示未读）|
| **设置** | ⚙️ | 打开设置面板 |

---

## 悬浮设置球

屏幕右下角有一个可**拖拽**的悬浮球，点击可快速打开设置面板。

- 可在「设置 → 界面选项」中**开启或关闭**悬浮球的显示
- 偏好保存在浏览器本地，刷新后不会重置

---

## 主内容标签页

| 标签 | 内容 |
|------|------|
| 🌾 **农场** | 作物种植格，点击种植/收割 |
| 👾 **怪兽** | 已捕捉怪兽列表，查看属性/出售 |
| 🗺️ **探索** | 选择区域手动探索，查看已解锁区域 |
| 🔬 **科技** | 科技树，消耗研究点解锁节点 |
| 🏗️ **建筑** | 建造/升级建筑 |

---

## 快捷键汇总

| 快捷键 | 功能 |
|--------|------|
| `S` | 手动存档 |
| `Esc` | 关闭当前弹窗 |

<div class="info">
📌 更多快捷键持续开发中，可在游戏内「设置 → 快捷键」中查看完整列表。
</div>

## 设置说明

设置面板（⚙️）包含以下配置项：

### 界面选项

| 选项 | 说明 |
|------|------|
| 悬浮设置球 | 开关右下角悬浮快捷入口，偏好本地持久化 |

### 显示设置

| 选项 | 说明 |
|------|------|
| 字体大小 | 调整全局字体，范围 12px ~ 18px |
| 语言 | 切换界面语言（当前支持：简体中文） |

### 数据管理

| 选项 | 说明 |
|------|------|
| 导出存档 | 将存档导出为文本字符串，可备份或迁移 |
| 导入存档 | 粘贴存档字符串恢复游戏进度 |
| 重置游戏 | **不可撤销**，清除全部进度重新开始 |

<div class="danger">
⛔ 重置游戏操作不可逆！执行前请先导出存档进行备份。
</div>

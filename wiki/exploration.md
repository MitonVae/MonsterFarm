# 🗺️ 探索系统

## 探索概述

探索是获取多样资源和捕捉怪兽的核心途径。通过在不同区域积累**探索进度**，玩家可以获得金币、食物、材料、研究点，以及遭遇并捕捉野生怪兽。

### 探索方式

| 方式 | 说明 |
|------|------|
| **手动探索** | 点击探索按钮，每次消耗能量，增加一定进度 |
| **自动探索** | 解锁探索营地后，怪兽可自动积累进度（无需能量）|

## 奖励与掉落

### 掉落类型

探索完成后，根据区域数据在以下范围内随机产出：

```
奖励 = random(min, max) × 区域系数 × 怪兽属性加成
```

| 资源 | 影响因素 |
|------|----------|
| 🪙 金币 | 区域 tier，泰坦巨人加成 +200% |
| 🌽 食物 | 区域 tier |
| 🪨 材料 | 区域 tier，灰烬魔偶/钢铁守卫加成 |
| 🔬 研究点 | 区域 tier，冰晶/天界使者/虚空行者加成 |

### 探索进度与能量

- 每次手动点击消耗固定能量（见区域列表）
- 每次点击获得 `progressPerClick` 范围内的随机进度
- 进度达到 100% 后自动结算奖励并重置进度

<div class="tip">
💡 将高敏捷（agility）怪兽分配到探索任务可显著加快进度积累速度。风精灵的探索速度 +40% 效果非常实用。
</div>

## 解锁条件速查

| 条件类型 | 说明 |
|----------|------|
| `coins` | 当前拥有金币数 ≥ 指定值 |
| `totalExplorations` | 历史总探索完成次数 ≥ 指定值 |
| `monsterCount` | 当前拥有怪兽数量 ≥ 指定值 |
| `materials` | 当前拥有材料数 ≥ 指定值 |
| `research` | 当前拥有研究点 ≥ 指定值 |
| `tech` | 已解锁指定科技节点 |
| `compound` | 以上条件的组合（需全部满足）|

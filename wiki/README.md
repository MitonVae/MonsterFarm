## 游戏简介

《怪兽农场》是一款以**增量经营**为核心的浏览器游戏。玩家从一片荒地起步，建造农场，收割作物，同时深入野外捕捉各类怪兽。

怪兽不仅是农场的劳力，也是探索世界的主力——带着它们去探索未知区域，收集更稀有的资源，解锁更高阶的科技，直到掌控这片土地的一切。

### 核心循环

```
种地收获资源 → 解锁科技 → 探索新区域 → 捕捉更强怪兽
       ↑                                        |
       └────────── 更高效的农场产出 ←──────────┘
```

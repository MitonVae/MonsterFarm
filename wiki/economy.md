# 💰 经济系统

## 金币来源

金币是游戏中最基础的流通货币，主要来源：

| 来源 | 说明 |
|------|------|
| 收割作物 | `产量 × 单价`，是早期主要来源 |
| 探索奖励 | 每次探索完成后随机获得 |
| 出售怪兽 | 将多余怪兽出售可回收部分金币 |
| 泰坦巨人加成 | 拥有传说级怪兽泰坦巨人，金币产出 +200% |

### 金币消耗

| 用途 | 说明 |
|------|------|
| 建造建筑 | 主要消耗来源，单次花费较大 |
| 升级建筑 | 每级升级费用递增 |
| 解锁科技 | 部分科技节点需消耗金币 |
| 怪兽维护 | 每只怪兽每 tick 扣除少量金币（除岩背龟维护费减半外）|

## 能量管理

**能量（⚡）** 是手动探索的消耗品，上限由建筑和科技决定。

### 能量恢复

- 自然恢复：每 tick 恢复固定量
- 食物消耗：消耗食物可快速补充能量
- 雷鸟加成：拥有雷鸟（Rare），能量恢复速度 +30%

### 能量消耗

| 区域层级 | 每次手动探索消耗 |
|----------|----------------|
| Tier 1 | 5 |
| Tier 2 | 8 ~ 10 |
| Tier 3 | 12 ~ 14 |
| Tier 4 | 15 ~ 18 |
| Tier 5 | 20 ~ 25 |

<div class="tip">
💡 能量不足时，自动探索仍可继续运作（自动探索不消耗能量）。合理搭配手动/自动探索是中期效率的关键。
</div>

## 科技树

科技树是解锁游戏新内容的核心通道。消耗**研究点**解锁节点，每个节点提供持久性的数值提升或内容解锁。

### 科技类别

| 类别 | 解锁效果 |
|------|----------|
| **农业科技** | 解锁高阶作物（advancedFarming、cropT3~T6）|
| **探索科技** | 降低探索能量消耗、提高捕捉概率 |
| **怪兽科技** | 提升怪兽成长速度、特性品质 |
| **经济科技** | 提升金币/材料收益率 |
| **研究科技** | 加速研究点产出 |

### 关键科技节点

| 科技 ID | 效果 | 解锁条件 |
|---------|------|----------|
| `advancedFarming` | 解锁 Tier 2 全部作物 | 研究点 ≥ 一定数量 |
| `cropT3` | 解锁向日葵/草药/棉花/甘蔗 | 依赖 advancedFarming |
| `cropT4` | 解锁火龙果/冰蕨/虚空菇/黄金小麦 | 依赖 cropT3 |
| `cropT5` | 解锁星辰果/月光叶/血玫瑰 | 依赖 cropT4 |
| `cropT6` | 解锁顶级作物三种 | 依赖所有前置科技 |
| `exploration` | 解锁高级探索区域 | 基础探索完成 |

<div class="info">
📌 科技树是线性依赖结构，建议优先推进农业科技主线，解锁高价值作物后再转向探索/怪兽分支。
</div>
